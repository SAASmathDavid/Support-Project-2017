    Example PreCalculus Problems

    Problem 1: Find the domain and range for `f(x) = 4 - x^2`.

    Problem 2: Find the domain and range for `f(x) = 5 root() (9 - x)`.

    Problem 3: Find the domain and range for `f(x) = (5x)/(x - 8)`.

    Problem 4: Find the domain for `f(x) = (4x - 20)/(root() (36 - x^2))`.

    Problem 5: If `f(x) = x^2 + 4` and `g(x) = root() (2x - 3)`, find `(f @ g)(x)` and `(g @ f)(x)`.

    Show Solutions

    Solution 1: `f` is defined for all values of `x` (since `f` is a polynomial), so the domain of `f` is `(-oo, oo)`. Since the graph of `f` is a parabola which opens downward with vertex at `(0, -4)`, the set of y-coordinates for the points on the graph of `f` consists of all y-values with `y <= 4`; so the range of `f` is the interval `(-oo, 4]`.

    Solution 2: `f` is defined where `9 - c >= 0` or `9 >= x`, so the domain of `f` is the interval `(-oo, 9]`. Since `root() (9 - x) >= 0`, `f(x) = 5 + root (9 - x) >= 5` for any `x` in the domain of `f`, so the range of `f` is contained in `[5, oo)`. If `y >= 5`: `f(9-(y – 5)^2) = 5 + root() (9 - (9 - (y - 5))) = 5 + root() ((y - 5)^2) = 5 + (y - 5) = y`
    (since `y - 5 >= 0`), so the range of `f` is actually equal to `[5, oo)`.

    Solution 3: `f` is defined for `x - 8 != 0` or `x != 8`, so the domain of `f` is given by `(-oo, 8) uu (8, oo)`. To find the range of `f` we must determine for which y-values the equation `y = (5x)/(x - 8)` has a solution for `x`. Multiplying both sides of this equation by `x - 8` gives:
    `(x - 8)y = 5x` or `yx - 8y = 5x`, so `yx - 5x = 8y rArr (y-5)x = 8y rArr x = (8y)/(5 - y)`. Therefore the equation `y = (5x)/(x - 8)` has a solution for `x` if `5 - y != 0` or `y != 5`, so the range of `f` is given by `(-oo, 5) uu (5, oo)`.

    Solution 4: `f` is defined wherever `36 – x^2 >= 0` (so the square root is defined) and `36 - x^2 != 0` (so the fraction is defined). Solving the inequality `36 - x^2 > 0` or `(6 - x)(6 + x) > 0` gives `-6 < x < 6`, so the domain off is `(-6, 6)`.

    Solution 5: `(f @ g)(x) = f(root() (2x-3)) = (root() (2x-3))^2 + 4 = 2x -3 + 4 = 2x + 1`, while `(g @ f)(x) = g(x^2 + 4) = root() (2(x^2 + 4) - 3) = root() (2x^2 +5)`.

    PreCalculus Definitions

    Asymptote A line or curve that approaches a given curve arbitrarily closely.
    Curve A continuous map from a one-dimensional space to an n-dimensional space. Loosely speaking, the word "curve" is often used to mean the function graph of a two- or three-dimensional curve.
    Determinant (1) A function that assigns a scalar to a square matrix (or, equivalently, its linear transformation). (2) The value of this function for a particular matrix. The matrix has an inverse if and only if its determinant is nonzero.
    Parametric Equations A set of equations that together express a set of quantities as explicit functions of a number of independent variables, which are known as parameters.
    Plane A two-dimensional surface defined by linear equations.
    Plane Curve A curve that lies in a single plane. A plane curve may be closed or open.
    Polar Coordinates A two-dimensional coordinate system in which points in two dimensions are given by an angle and a distance from the origin.
    Rational Function A function that can be written as the quotient of two polynomials.
    Reflection The operation of exchanging all points of a mathematical object with their mirror images.
    Rotation The turning of an object or coordinate system about a fixed point.
    Rotation Matrix A matrix that corresponds to the linear transformation of a rotation.
    Scalar A value (such as a measurement) that has only magnitude but not direction. This contrasts with a vector, which has direction as well as magnitude.
    Spherical Coordinates A coordinate system in which points in three-dimensional space are given by two angles and a distance from the origin.
    Tangent Line A line that touches but does not cross a curve at a given point.
    Translation A transformation consisting of a constant shift with no rotation or stretching.
